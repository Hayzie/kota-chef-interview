### Installation

Vue
  1. Install NPM packages
   ```sh
   npm install
   ```
  2. Add the VITE_API_BASE_URL= 'ENTER YOUR BASE URL eg. http://localhost:300' in you dotenv file
  3. Run development
   ```sh
   npm run dev
   ```
Server
   1. Install packages
   ```sh
   bun install
   ```
   2. Add this in your dotenv DATABASE_URL='YOUR DB URL', and JWT_SECRET='YOUR TOKEN SECRET'
   3. Run development
   ```sh
   bun run dev
   ```
